// ripgrep //

Fast recursive search using rg

Wrap ripgrep as a Tier 2 orcai plugin via sidecar YAML. Runs rg --json and pipes structured output into pipelines.

Sidecar Config

name: ripgrep
command: rg
args:
  - --json
  - "{{query}}"

Usage

steps:
  - id: search
    provider: ripgrep
    input:
      query: "TODO"
