┌─ ECOSYSTEM

Provider plugins ship with gl1tch — no install needed.
Community plugins install with one command. Build your own with a binary and a YAML manifest.

│ PROVIDERS — bundled, zero install, ready in any pipeline step
ollama bundled

Run any local model via Ollama. Ships with gl1tch — the default provider for all internal ops.

binary glitch-ollama
models any model pulled to Ollama
docs ollama.com
claude bundled

Claude AI assistant via the claude CLI.

binary glitch-claude
models opus-4-6 · sonnet-4-6 · haiku-4-5
docs claude.ai/code
gemini bundled

Google Gemini via the gemini CLI.

binary glitch-gemini
models 2.5-pro · 2.5-flash · 2.0-flash · 1.5-pro
docs ai.google.dev/gemini-api/docs/gemini-cli
codex bundled

OpenAI Codex coding agent.

binary glitch-codex
models codex-mini-latest · o4-mini · o3
docs github.com/openai/codex
github-copilot bundled

GitHub Copilot suggestions via the gh CLI.

binary glitch-github-copilot
docs cli.github.com
opencode bundled

Agentic AI coding via opencode. Pairs with any provider/model string.

binary glitch-opencode
models any provider/model (e.g. ollama/llama3.2)
docs opencode.ai
