// What Is GLITCH //

orcai is a Go BubbleTea TUI that turns your terminal into an AI command center. Manage tmux sessions from a BBS-style sidebar, wire up any CLI tool as a plugin via gRPC or sidecar YAML, and run multi-step AI agent pipelines across Claude, Ollama, GitHub Copilot — all from one dark, dense, beautiful interface.

┌─ TMUX SESSIONS ─┐

Manage AI agent workspaces from a BBS-style sidebar TUI. Spawn, switch, and kill tmux sessions without leaving your workflow. Each session gets its own pane layout and context.

┌─ PLUGIN SYSTEM ─┐

gRPC plugins + sidecar YAML. Any CLI tool becomes a pipeline node. Tier 1 native Go plugins for speed. Tier 2 wrappers for everything else — ripgrep, jq, ffmpeg, your own scripts.

┌─ AGENT PIPELINES ─┐

YAML pipelines run Claude, Ollama, and Copilot at scale. Chain tools together with typed inputs and outputs. The prompt builder sidebar lets you compose on the fly.
