# TDF Font Licenses

## About TDF Fonts

TDF (TheDraw Font) files are a binary font format from the BBS era, used to
create ANSI block-letter art. Real TDF font files can be sourced from:

- **ACiD Productions** — http://www.acid.org/
- **iCE Advertisements** — creators of many classic BBS fonts
- **TheDraw** — the original ANSI editor (freeware) that defined the format

## Fonts in This Repository

### placeholder.tdf

The `placeholder.tdf` file is a **synthetic test file** generated solely to
validate that the TDF parser works correctly. It contains only the TDF magic
header and a minimal font record with no character blocks.

**This file has no copyright and is dedicated to the public domain.**

## Adding Real TDF Fonts

Real TDF font files can be added to this directory. Before distributing any
TDF font, verify that you have permission from the original artist or that
the font is freely redistributable. Many classic BBS fonts were released as
freeware or public domain; when in doubt, contact the artist or their group.
